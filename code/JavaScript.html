# Promise

Promise对象是一个构造函数,用来生成Promise实例.用来解决异步编程的一种方案

两个参数，三种状态

两个参数:

  1. resolve 在异步操作成功的调用,并能将异步操作的结果作为参数传递出去。

    pending 变为fulfilled

  2. reject 在异步操作失败时调用，并能将异步操作报出的错误，作为参数传递出去。

    pending 变为 rejected

三种状态:

  1. pending 进行中
  2. fulfilled 已成功
  3. rejected 已失败

当 pending 变为 fulfilled 或 rejected 两种某一种时，状态就已经凝固了,不会再变了,会一直保持这个结果,这时就成为 resolved(已定性)。如果改变已经发生，再对 Promise 对象添加回调函数，也会立即得到这个结果。事件特性时，如果你错过了它,再去监听,是得不到结果的。

两个特点：

对象的状态不受外界影响。

一旦状态改变，就不会再变，任何时候都可以得到这个结果。

缺点：

首先，无法取消Promise，一旦新建它就会立即执行，无法中途取消。

其次，如果不设置回调函数，Promise内部抛出的错误，不会反应到外部。

使用方式

  let img = new Image();

let express=require("express");
let app=express();
//解构赋值
let {readFile}=require("fs");
//promise化
read=url=>{
    //调用read时返回 Promise
    return new Promise(resolve=>{
      //读取文件
        readFile(url,(err,data)=>{
          //成功后返回resolve回调,文件内容以参数形式传入
            resolve(data);
        })
    })
}

app.get("/goodList",async (req,res)=>{
  									//路径传参,返回元素内容 加上await等这一步运行完,运行下一步
    let data=await read("./data.json");
  		//转换格式
    res.json(JSON.parse(data.toString()))
})
